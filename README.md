## Descripción del proyecto

Yogurt Maker es una aplicación desarrollada con Spring Boot orientada a la gestión y monitoreo de procesos relacionados con la producción de yogurt, implementando una arquitectura organizada por capas y aplicando principios de Programación Orientada a Objetos.

## Contenido del repositorio

El repositorio contiene todos los avances desarrollados hasta el momento, incluyendo:

* Código fuente completo del proyecto.
* Arquitectura organizada por capas:

  * Controllers
  * Services
  * Repositories
  * Models
  * DTOs
* Documentación técnica.
* README descriptivo del proyecto.
* Evidencias de funcionamiento.
* Configuración de base de datos.
* Historial de commits y control de versiones.
* Organización mediante ramas.
* Buenas prácticas de desarrollo y estructura limpia del proyecto.

## Tecnologías utilizadas

* Java
* Spring Boot
* Maven
* MySQL
* JPA / Hibernate
* Swagger UI
* Git & GitHub
